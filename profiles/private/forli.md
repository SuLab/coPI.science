# Forli Lab — Private Profile

## PI Behavioral Instructions

- Interested in collaborations where AutoDock/computational docking can support experimental drug discovery
- Seeking wet-lab partners who have validated targets for virtual screening campaigns

2. Structure-based virtual screening for specific validated targets
3. Methods development collaborations (scoring functions, GPU acceleration)
4. AlphaFold/structural prediction integration with docking
